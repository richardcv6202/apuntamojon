📖 Manual de Usuario

📌 ¿Qué es ApuntaMOJÓN?

Aplicación para control de partidas de dominó donde el perdedor completa la palabra MOJÓN.

📏 Escala de tantos a letras

  • <26 tantos → 1 letra
  • 26-50 tantos → 2 letras
  • 51-75 tantos → 3 letras
  • 76-100 tantos → 4 letras
  • 101-125 tantos → 5 letras
  • ≥126 tantos → 6 letras (MOJÓN completo)

📜 Formación de MOJÓN

  • 1° pérdida → M
  • 2° pérdida → MO
  • 3° pérdida → MOJ
  • 4° pérdida → MOJO
  • 5° pérdida → MOJÓ (O → Ó)
  • 6° pérdida → MOJÓN → PIERDE

🎮 Cómo anotar una mano

  1. Presiona "Anotación"
  2. Marca "Un jugador se pegó" si aplica
  3. Selecciona quién cerró o pegó
  4. Selecciona el/los perdedores
  5. Ingresa los tantos acumulados
  6. Presiona "Aceptar"

📊 Significado de estadísticas

  • +Gorda: Máximo de tantos con que perdió
  • % Mojón: Porcentaje de partidas perdidas
  • Letras acumuladas: Total de letras recibidas

💾 Gestión de datos

  • Exportar: Guarda copia de seguridad
  • Importar: Restaura datos desde archivo
  • Limpiar BD: Elimina todos los datos (contraseña: mojon)

❓ Preguntas Frecuentes

  • ¿Puedo jugar sin conexión? Sí, es una PWA que funciona offline.
  • ¿Cómo instalo la app? Abre la URL en Chrome → "Añadir a pantalla de inicio".
  • ¿Qué pasa si cierro la app? La partida se guarda automáticamente.
  • ¿Los datos están en la nube? No, solo en tu dispositivo. Exporta para respaldo.
